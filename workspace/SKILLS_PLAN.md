# SKILLS_PLAN.md - Skills for Dave & Moon-seop

## Essential Skills (Must Have)

1.  **web_search / web_fetch** (기본 내장)
    -   **용도:** 후보자 서칭, 기업 정보 조사, 뉴스 트래킹, 맛집/여행 정보 검색.
    -   **상태:** 이미 사용 가능.

2.  **summarize** (설치 필요)
    -   **용도:** 긴 JD(Job Description), 뉴스 기사, 유튜브 영상 요약해서 문섭이 시간 아껴주기.
    -   **상태:** `/opt/homebrew/lib/node_modules/openclaw/skills/summarize`에 있음.

3.  **gemini** (설치 필요)
    -   **용도:** 이메일 초안 작성, 후보자 추천사 작성, 복잡한 데이터 정리. (지금도 똑똑하지만, 긴 글 쓸 때 유용)
    -   **상태:** `/opt/homebrew/lib/node_modules/openclaw/skills/gemini`에 있음.

4.  **weather** (설치 필요)
    -   **용도:** 미팅 날짜 잡을 때 날씨 체크, 출장/여행지 날씨 확인.
    -   **상태:** `/opt/homebrew/lib/node_modules/openclaw/skills/weather`에 있음.

5.  **nano-pdf** (설치 필요)
    -   **용도:** 이력서(PDF) 내용을 텍스트로 뽑거나 분석할 때.
    -   **상태:** `/opt/homebrew/lib/node_modules/openclaw/skills/nano-pdf`에 있음.
